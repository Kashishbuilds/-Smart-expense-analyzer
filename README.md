# Smart Expense Analyzer 💰

Smart Expense Analyzer is a machine learning-based web app built using Streamlit to track, analyze, and predict expenses.

## Features

* Track daily expenses
* Expense visualization charts
* Smart insights
* Chatbot support
* Database storage

## Tech Stack

* Python
* Streamlit
* Machine Learning
* SQLite
* Pandas

## Project Structure

```
Smart-expense-analyzer/
│
├── app.py
├── requirements.txt
├── utils/
├── pages/
├── models/
```

## How to Run Locally

1. Install dependencies

```
pip install -r requirements.txt
```

2. Run the app

```
streamlit run app.py
```
